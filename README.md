# 🏛️ Government Central Helpline — AI-Powered Grievance System

AI-powered multi-channel grievance redressal system for government helplines.

## Features
- Multi-Channel Input (Phone, WhatsApp, Email, Web, Walk-in)
- AI Auto-Categorization & Smart Priority Scoring
- Sentiment Analysis & Complaint Tracking
- Officer Panel & Supervisor Dashboard with AI Insights

## Run Locally
```bash
pip install -r requirements.txt
streamlit run helpline_app.py
```
